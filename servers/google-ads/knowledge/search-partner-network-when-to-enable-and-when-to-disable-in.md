## Key Tactics
- **Disable search partners by default on new campaigns** — Google forces search partners on during campaign setup, but you should uncheck "Include Google Search Partners" in campaign settings to maintain control over traffic quality.
- **Search partners lack transparency** — You receive zero visibility into where clicks originate when search partners are enabled, making it difficult to assess traffic quality and performance.
- **Search partners increase bot and spam traffic risk** — The network is prone to generating low-quality clicks, particularly from park domains and non-Google websites, which can waste budget.
- **Enable search partners only with trained smart-bidding campaigns** — If you're using TOAS (Target ROAS) or TCPA (Target CPA) with high-quality historical conversion data, the algorithm can filter out poor traffic, making search partners safer to test.
- **Use network segmentation to monitor search partner performance** — Access the Segments menu, select "Network," and filter by "Search Partners" to see if this channel is actually driving quality conversions before deciding whether to keep it enabled.

## Summary
Search partner network is disabled by default in new campaigns for good reason—it offers zero transparency and attracts bot/spam traffic. Only enable search partners if you're running smart-bidding campaigns (TOAS/TCPA) with strong conversion training data, as the algorithm can then filter low-quality clicks effectively.
